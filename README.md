# netease_api_parse

## 使用

需配合 **[网易云音乐 NodeJS 版 API](https://neteasecloudmusicapi.vercel.app)** 食用。

使用以下命令运行：

``` shell
java -jar netease_music_api_test-1.0-SNAPSHOT.jar
```

或者 clone 项目，使用 IDE 运行。
